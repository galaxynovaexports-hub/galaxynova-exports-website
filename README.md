## 🌌 About

This is the official landing page for **STELORA™**, the premium GI-tagged aromatic rice brand by **Galaxynova Exports & Trading (OPC) Private Limited**, West Bengal, India.

The design is inspired by the **Aurora Borealis** — mirroring the STELORA packaging aesthetic with deep teal/navy tones, glowing green aurora effects, and a celestial star motif.

## 🚀 Deployment

### Option 1 — GitHub Pages (Recommended)
1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://yourusername.github.io/stelora-website/`

### Option 2 — Direct hosting
Upload `index.html` to any static web host:
- [Netlify Drop](https://app.netlify.com/drop) — drag & drop, instant live URL
- [Vercel](https://vercel.com) — connect GitHub repo
- Any cPanel/FTP web hosting (upload to `public_html/`)

### Option 3 — Local preview
```bash
# Simply open in browser:
open index.html

# Or serve locally with Python:
python3 -m http.server 8080
# Then visit http://localhost:8080
```
